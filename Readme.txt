Простенький корпоративный органайзер, созданный с нуля в те времена, когда о Service Desktop мало что было известно. 
Есть возможность создавать задачи подчиненным из списка, переадресовывать задачи, устанавливать приоритет, важность, 
указывать процент выполнения, просматривать результирующее дерево задачи.
